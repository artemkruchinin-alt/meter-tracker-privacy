Политика конфиденциальности

Дата последнего обновления: 3 февраля 2025 года

Настоящая Политика конфиденциальности описывает, как мобильное приложение «Учёт ЖКУ» (далее — «Приложение») обрабатывает информацию пользователей.

1. Сбор информации

Приложение не собирает, не передаёт и не хранит персональные данные пользователей на внешних серверах.

Все данные, которые вы вводите в Приложение (показания счётчиков, информация о квитанциях, фотографии), хранятся исключительно на вашем устройстве в локальной базе данных.

2. Какие данные обрабатываются локально

Приложение сохраняет на вашем устройстве:

  • Информацию о счётчиках (название, тип, серийный номер)
  • Показания счётчиков и даты их внесения
  • Фотографии показаний счётчиков
  • Информацию о квитанциях (сумма, период, статус оплаты)
  • Данные, полученные при сканировании QR-кодов квитанций

3. Разрешения приложения

Приложение запрашивает следующие разрешения:

  • Камера — для сканирования QR-кодов квитанций и фотографирования показаний счётчиков. Фотографии сохраняются только на вашем устройстве.
  • Хранилище — для сохранения фотографий показаний в память устройства.

Приложение не использует разрешения для сбора или передачи данных третьим лицам.

4. Передача данных третьим лицам

Приложение не передаёт никакие данные третьим лицам, рекламным сетям или аналитическим сервисам.

Приложение не требует подключения к интернету для работы и функционирует полностью автономно.

5. Безопасность данных

Все данные хранятся в защищённой области памяти вашего устройства, доступной только данному Приложению. При удалении Приложения все данные автоматически удаляются.

6. Удаление данных

Вы можете удалить все данные следующими способами:

  • Через настройки Приложения — функция «Очистить все данные»
  • Удалив Приложение с устройства
  • Через настройки Android — «Приложения» → «Учёт ЖКУ» → «Очистить данные»

7. Дети

Приложение не предназначено для сбора данных от детей младше 13 лет и не собирает сознательно такую информацию.

8. Изменения политики

Мы можем обновлять данную Политику конфиденциальности. Актуальная версия всегда доступна по данной ссылке. Рекомендуем периодически проверять эту страницу.

9. Согласие

Используя Приложение, вы соглашаетесь с условиями данной Политики конфиденциальности.
